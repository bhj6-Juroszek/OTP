Requirements:

Java Platform (JDK) 8
Apache Maven 3.x
Internet connection
Gmail account with enabled external applications access

Quick start:

-change properties of mail sender bean (username and password) in resources/Mail.xml,
otherwise You won't have access to mail sender object and app will crush
-mvn package
-java -jar target/executable-1.jar
-Point your browser to http://localhost:8181

IMPORTANT:
Application is in development process. Change from VAADIN GUI to http views in progress. Currently application is in one big mess ;)
